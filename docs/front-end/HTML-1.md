## 1.1. 概念

*HTML*文档通常以**文档声明**开始，该声明的作用是**帮助浏览器确定其解析和显示的*HTML*文档类型**。

```html
<!DOCTYPE html>
```

* 文档声明必须在第一行、且顶格显示
* 大小写不敏感

如果有内容放在其前面，比如批注、*XML*声明等，会令*IE9*或者更早期的浏览器触发**怪异模式**（混杂模式）

文档声明并非一个*HTML*标签。它是一个"信息"，告诉浏览器期望的文档类型。
