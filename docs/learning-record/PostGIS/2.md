## 2.1. 问题

### 2.1.1. 备份/恢复数据提示

###### 2.1.1.1. 提示信息

```
Utility file not found. Please correct the Binary Path in the Preferences dialog
```

翻译: `找不到实用程序文件。请在“首选项”对话框中更正二进制路径`

###### 2.1.1.2. 解决办法

依次打开: `File - Preferences - Paths - Binary paths`, 选择安装目中的`bin`文件

![alt text](./images/2/image1.png)
